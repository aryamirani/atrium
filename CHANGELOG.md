# Changelog

Atrium's changelog is generated automatically for each tagged release and
published on the GitHub Releases page. Commits are grouped into **Features** and
**Fixes** from their [Conventional Commit](https://www.conventionalcommits.org)
prefixes (see [CONTRIBUTING.md](CONTRIBUTING.md)).

➡️ **See the full, per-version changelog at:**
<https://github.com/ZviBaratz/atrium/releases>

This file intentionally points to Releases rather than duplicating the history,
to avoid drift between a hand-maintained changelog and the generated one.
